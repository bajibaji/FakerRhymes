# 废壳押韵机器人 FakerRhymes

蟹不肉，**FakerRhymes** 是一款专为喜欢押韵的沙雕设计的中文押韵生成器。它能够自动分析输入词汇的拼音、声调和韵母，并基于强大的内置词库或 AI 模型，生成同声调、同韵母的押韵字串。无论是写歌词、写诗还是文案创作，它都能为你提供丰富的灵感。

---

## 🚀 快速开始

### 1. 初始化词库
首次打开软件时，请务必点击界面上的 **"📚 加载词库"** 按钮。
*   程序会从在线/本地资源加载 7000+ 常用汉字数据。
*   **离线可用**：数据加载完成后会自动缓存到本地（localStorage），下次打开无需再次加载。

### 2. 生成押韵
1.  在 **"请输入要押韵的词语"** 文本框中输入目标词汇（例如："废壳"）。
2.  点击 **"生成押韵"** 按钮（或使用快捷键 `Ctrl + Enter`）。
3.  程序将在下方显示生成的押韵结果。

---

## 🎛️ 功能详解

### 1. 韵脚宽松度调节
通过拖动滑块，你可以控制押韵的严格程度，以适应不同的创作需求：

*   **🎯 严格 (0.0)**
    *   **规则**：要求**韵母完全相同**且**声调完全一致**。
    *   **适用**：追求完美工整的对仗或严谨的格律诗。
*   **⚖️ 中等 (0.5)**
    *   **规则**：**声调必须一致**，但**韵母要求放宽**。
    *   **细节**：兼容了模糊音（如前鼻音 `an` 与后鼻音 `ang`，卷舌音 `zh` 与平舌音 `z` 等），扩大了匹配范围。
*   **🌊 最松 (1.0)**
    *   **规则**：**忽略声调限制**，且**韵母要求进一步放宽**。
    *   **适用**：现代说唱（Rap）、自由诗或寻找广泛的灵感。

### 2. 🎼 平仄过滤
当**韵脚宽松度**设为 **"最松 (1.0)"** 时，界面会出现平仄过滤选项：
*   **全部**：显示所有声调的结果。
*   **平 (Ping)**：只显示一声（阴平）和二声（阳平）的字。
*   **仄 (Ze)**：只显示三声（上声）和四声（去声）的字。
> *注：此功能对于需要讲究平仄格律的古典诗词创作非常有用。*

### 3. 🤖 AI 押韵模式 (Gemini)
除了内置词库，你还可以利用 Google Gemini AI 的强大能力来生成更具创意、更符合语境的押韵词。

*   **开启方法**：勾选界面上的 **"AI 押韵模式"** 复选框。
*   **配置**：点击旁边的 **设置图标 (⚙️)**，填入你的 **Gemini API Key**。
*   **网络设置**：支持配置 HTTP/HTTPS/SOCKS 代理，方便国内用户使用。
*   **特点**：AI 模式不再局限于词库，能生成更多新造词、流行语或意象化的表达。

### 4. 📝 自定义词库
觉得内置词库不够用？你可以添加自己的专属词汇！
1.  点击右上角的 **"自定义词库"** 链接。
2.  在文本框中输入你想要添加的词语（用空格、逗号或换行分隔）。
3.  点击 **"保存到浏览器"**。
4.  返回主页，这些词汇就会立即参与到押韵计算中。

### 5. 🔍 结果分析与微调
生成结果后，界面下方会显示详细的分析表格：
*   **音调选择表**：列出原词中每个字的拼音、声调和韵母。
*   **手动修正**：如果自动识别的拼音或韵母有误（例如多音字），你可以直接在表格中修改，程序会根据修改后的条件重新匹配。

## 💡 常见问题 (FAQ)

**Q: 点击"生成押韵"后显示乱码或没有结果？**
A: 这通常是因为条件太严格，词库中找不到完全匹配的词。请尝试：
1.  将 **"韵脚宽松度"** 滑块向右拖动（调松）。
2.  检查输入的词语是否过于生僻。

**Q: 加载词库很慢怎么办？**
A: 词库加载取决于网络速度和设备性能。好消息是只需加载一次，后续使用都是秒开（离线缓存）。

**Q: AI 模式报错？**
A: 请检查：
1.  API Key 是否正确。
2.  网络是否通畅（如果需要代理，请在设置中正确填写代理地址，格式如 `http://127.0.0.1:7890`）。
